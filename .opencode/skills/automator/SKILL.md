# Automator Bridge

Execute AppleScript and JavaScript for Automation (JXA) on the host Mac.

## Usage

- AppleScript: `automator.run_applescript('tell application "Finder" ...')`
- JXA: `automator.run_jxa('Application("Finder")...')`
